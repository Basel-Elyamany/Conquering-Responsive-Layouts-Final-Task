- 🎨 Clean and modern UI
- 🔤 Google Fonts integration
- 📊 Structured sections (header, intro, content, footer)

## 📱 Responsive Design

The layout adapts using media queries:

### 🔸 600px and above
- Navigation bar appears
- Layout switches to flex rows
- Improved spacing and typography

---

### 🔸 1000px and above
- Two-column layout in middle section
- Sidebar steps stacked vertically
- Enhanced spacing and alignment

---

## 📜 Key CSS Concepts Used

- Flexbox for layout control
- Media queries for responsiveness
- `clamp()` and `min()` for fluid typography
- Mobile-first approach
